# 本轮可行性验证边界

## 已实现并实跑

- 确定性合成 OHLCV 夹具、SHA-256 和数据元信息；
- 日期、维度、价格关系、缺失值与训练/验证/测试切分校验；
- 仅使用 `t` 及以前信息的收益、EWMA 波动率、样本/收缩协方差、相关性、流动性代理、信用代理和资产回撤；
- 训练期统计量拟合的特征标准化；
- 战略锚点的局部 5 个百分点邻域筛选；
- 共享编码器、方向头和规模头的 NumPy 前向验证；
- 零和、L1 归一化方向与有界 `alpha`；
- 权重范围、战略偏离、换手和组合风险的确定性凸投影；
- 动态交易成本、调仓前漂移持仓、财富、峰值和回撤更新；
- PPO 训练实现和非 RL 的投影梯度直接优化基线；
- 单步人工样例、固定随机种子、约束和因果性自动测试；
- 合成夹具端到端运行与标准 JSON 报告。

## 已完成的真实数据能力

- 统一的 `MarketDataProvider` 协议，以及 `yfinance`、本地 CSV/Parquet 和
  合成夹具 Provider；任何在线失败、字段缺失或资产缺失都会显式报错，绝不
  回退到合成数据；
- 原始长表 OHLCV、资产元数据、数据集清单、质量报告和 SHA-256 清单的
  不可覆盖冻结；训练按 `dataset_id` 只读本地冻结文件；
- 调整后收盘价收益、原始 `close × volume` 近似成交额、日频滚动特征、
  每周最后有效交易日和相邻决策日周收益的统一因果处理；
- 样本/收缩风险矩阵、固定/扩展/滚动三类切分，以及每轮仅用训练区间拟合
  的 scaler；
- 全量战略锚点枚举、训练/验证筛选、近似最优集合、简单性和邻域稳定性；
- 真实/合成共用的 `ProcessedMarketDataset`、投资环境和
  `experiments/train_ppo.py`；
- 训练区间 PPO 拟合，以及 train/validation/test 三段确定性回放、完整指标、
  逐期审计序列和战略锚点基准；
- 多随机种子独立输出、断点续跑与均值/标准差/范围汇总入口；
- 数据基础 JSON/PNG 报告以及真实环境连续三步计算轨迹；
- Provider、冻结校验和、OHLC、因果性、训练期 scaler、切分、协议一致性、
  真实环境回放和失败不回退的离线自动化测试。

## 当前冻结与实跑状态

- 已冻结真实数据集 `real_us_multi_asset_v1`：10 项美国市场 ETF，共
  46,670 条原始长表记录，公共日期范围为 2008-01-02 至 2026-07-22；
  原始文件和处理文件校验和均已复算通过；
- 已生成真实数据基础 JSON/PNG 报告，并在配置的 5% 战略搜索边界内全量
  评估 11,257 个可行锚点；
- 已完成一次真实数据技术 smoke run：`fixed_v1`、收缩协方差、CPU、
  随机种子 42、128 个 PPO 环境步；模型、训练日志和连续三个调仓时点的
  逐步计算保存在 `reports/real_us_multi_asset_v1/fixed_v1/ppo_smoke/`；
- 已额外实跑新的端到端评估入口：它只在训练区间更新 PPO 参数，并成功生成
  train、validation、test 的 PPO/战略锚点汇总指标与逐期结果，三段硬约束
  违反次数均为 0；
- 该 smoke run 只证明工程链路能够进入并完成 PPO 更新，不是正式 PPO
  训练，不构成真实市场有效性结论。

## 明确未完成

以下内容不能由本轮 CPU 合成夹具验证替代：

- GPU 上的 PPO 多随机种子正式训练；
- 滚动或扩展窗口的重复 PPO 训练与完全样本外评价；
- 全部基准和消融矩阵；
- 0/2/5/10/25bp 及压力成本情景的正式结果；
- SAC 和真正多步 MPC 对照；
- 统计显著性、真实经济结论和阶段二外部风险数据。

因此，当前 `passed` 仅表示“机制链路能运行且本地测试通过”，不表示策略有效，更不表示阶段一已通过研究验收。

项目已经形成“真实OHLCV构造与冻结 → 因果特征工程 → 战略锚点筛选 →
投资环境 → PPO训练 → train/validation/test 回放 → 多 seed 汇总”的真实
数据技术闭环。
